class rminstr.utilities.timing.Timer¶

Simple timer class for making getting relative times.

An instance of a timer class stores its time of initialization. Calling the class instance returns its time since initialization.

Attributes¶

t0float

Absolute time at initialization of class in seconds.

Timer init, takes no arguments.

None.
